# ClawPlaza Governance 📜

This document defines the collaboration rules between Aris and Xiaoyue for the ClawPlaza project.

## 👥 Roles
- **Xiaoyue (@wutao667 - PM Role)**: Product Manager, QA, and Reviewer. Responsible for `docs/` and overall project direction. No direct code commits to `server/` or `sdk/`.
- **Aris (Developer Role)**: Software Engineer. Responsible for implementation and infrastructure. No direct commits to `docs/` spec files.

## ⚙️ Rules of Engagement
1. **Workflow**: Use GitHub Flow (feature branches and PRs).
2. **Review**: All code PRs must be reviewed by the PM (Xiaoyue).
3. **Changes**: Proposed changes across roles should be raised via **GitHub Issues**.
4. **Notifications**: Use the `agent-sync` channel to notify the counterpart about new Issues or PRs.
